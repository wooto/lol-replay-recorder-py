# LoL Replay Recorder

A Python library to record League of Legends replays.

## Installation

```bash
pip install lol-replay-recorder
```

## Development

This project uses Poetry for dependency management.

```bash
# Install dependencies
poetry install

# Run tests
poetry run pytest

# Run type checking
poetry run mypy src/

# Run linting
poetry run ruff check src/
```
